## 🚀 Desktop Tool Features

The included **FiveM-Tool-Desktop.exe** helps with common setup and maintenance tasks:

1. Run `FiveM-Tool-Desktop.exe`.
2. The app will scan for your **GTA V** and **FiveM** installation folders.
3. Choose the folder manually if auto-detection is not available.
4. Open the **Settings Manager** to create or restore backups.
5. Use **Performance Presets** to apply local configuration changes.
6. Use **Cache Cleaner** to clear temporary files.
7. Launch your preferred client setup directly from the desktop tool.

> ℹ️ This tool is focused on desktop management, configuration, backup, and convenience for FiveM and GTA V users.

## ❗ Troubleshooting

| Problem | Solution |
|---------|----------|
| App does not detect GTA V folder | Select the installation directory manually |
| FiveM path is missing | Browse to the correct client folder in settings |
| Settings are not applied | Restart the app and verify folder permissions |
| Backup cannot be created | Run the app with the required file access permissions |
| Client issues after changes | Restore the previous backup from the Backup Utility |
| Temporary files keep returning | Run Cache Cleaner again after closing the client |
